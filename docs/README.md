## 🔐 安全设计要点

1. **认证:** JWT Token,有效期7天
2. **密码:** BCrypt加密存储
3. **敏感数据:** AES加密 (身份证号、手机号)
4. **传输:** HTTPS (生产环境)
5. **防护:** SQL注入、XSS、CSRF防护
6. **限流:** 登录失败5次锁定30分钟
